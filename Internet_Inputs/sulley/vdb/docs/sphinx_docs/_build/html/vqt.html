Add a dynamic menu item which will populate on-hover by asking the specified callback for a list of actions.

Example:

def callback(name=None):

if name == None:
return (‘one’, ‘two’, ‘three’)

print(‘SELECTED: %s’ % name)

m.addDynMenu( callback )
